What is it?

A lightweight PWM library port of Adafruit's Adafruit_SoftServo for AtTiny85 micro-controllers.

What differs from original?

refresh encapsulation

This library encapsulates the refresh cycle so you do not have to manually build the signal handler each time you use it. The downside to that, it uses slightly more memory than the original. The extra space consists of a globally scoped struct to hold the pin information and provide the refresh functionality.

overloaded write()

If you prefer to write in degrees (say using a servo), you can pass an uint8_t degree value to write. Or if you choose to work on duty cycles, you can pass a float duty cycle between 0 and 1.

Who can use it?

Works with servos, actuators, and really anything that needs a PWM signal. Duty cycle is built on 20 microsecond intervals.

How to download from github:

$ cd your_library_path
$ git clone https://github.com/anthonykulis/attiny85_pwm.git <optional-new-name>

How to use:

Include:

#include "attiny85_pwm.h"

Instantiate:

//assuming PIN_ONE defined...
Attiny85_PWM myPwm(PIN_ONE);

Write:

// degrees?
myPwm.write(42);

// optionally, duty cycle
myPwm.write(.42);
